1. PENGERTIAN CONSTRUCTOR

Constructor adalah metode khusus dalam kelas yang digunakan untuk menginisialisasi objek saat dibuat. Di Dart, constructor tidak memiliki tipe kembalian dan nama sama dengan nama kelasnya.

2. JENIS-JENIS CONSTRUCTOR

  • Default Constructor: Constructor tanpa parameter (otomatis dibuat jika tidak ada constructor lain). Contoh: Mahasiswa();
  • Parameterized Constructor: Constructor dengan parameter untuk mengisi nilai atribut langsung. Contoh: Mobil(this.merk, this.tahun);
  • Named Constructor: Constructor dengan nama khusus untuk membuat objek dengan cara berbeda. Contoh: Mahasiswa.dariMap(data);
  • Constant Constructor: Digunakan untuk membuat objek immutable dengan keyword const. Contoh: const Warna(255, 0, 0);
  • Constructor dengan Initializer List: Menginisialisasi atribut sebelum body constructor dijalankan. Contoh: Titik(double x, double y) : x = x.abs(), y = y.abs();

4. PENGERTIAN ENKAPSULASI

Enkapsulasi adalah konsep menyembunyikan detail implementasi data dengan membuat atribut menjadi private (dengan awalan _), lalu mengatur aksesnya melalui getter dan setter.

Tujuan enkapsulasi:

  • Menyembunyikan detail implementasi
  • Melindungi data dari perubahan tidak diinginkan
  • Membuat kode lebih mudah dipelihara
