Kelas dalam Dart

1. Kelas Buku
Penjelasan Kelas:
Merepresentasikan buku dengan atribut judul.
Menggunakan parameterized constructor untuk inisialisasi objek saat dibuat.
Status / Perbaikan:
Kode sudah benar, menampilkan judul buku.
2. Kelas Mobil
Penjelasan Kelas:
Merepresentasikan mobil dengan atribut merk dan tahun.
Memiliki parameterized constructor, named constructor baru() dan dariJson().
Status / Perbaikan:
Kode sudah benar, menampilkan detail data mobil.
3. Kelas Warna
Penjelasan Kelas:
Merepresentasikan warna dengan atribut RGB.
Menggunakan constant constructor untuk membuat objek immutable.
Status / Perbaikan:
Kode sudah benar, menampilkan nilai komponen warna.
4. Kelas Titik
Penjelasan Kelas:
Merepresentasikan koordinat dengan atribut x dan y.
Menggunakan initializer list untuk memproses nilai sebelum objek dibuat.
Status / Perbaikan:
Kode sudah benar, menampilkan koordinat positif.
5. Kelas Mahasiswa
Penjelasan Kelas:
Merepresentasikan data mahasiswa dengan atribut nama, NIM, jurusan, dan kelas.
Memiliki default constructor dan named constructor dariMap() serta tanpaNama().
Status / Perbaikan:
Kode sudah benar, menampilkan detail informasi mahasiswa.
6. Kelas Person
Penjelasan Kelas:
Merepresentasikan individu dengan atribut nama dan umur.
Menerapkan enkapsulasi dengan getter dan setter untuk atribut private.
Status / Perbaikan:
Kode sudah benar, menampilkan perubahan data nama.
7. Kelas Produk
Penjelasan Kelas:
Merepresentasikan barang dengan atribut nama dan harga.
Menerapkan enkapsulasi dengan validasi pada setter.
Status / Perbaikan:
Kode sudah benar setelah memperbaiki operator >=, menampilkan validasi data.
8. Kelas User
Penjelasan Kelas:
Merepresentasikan akun dengan atribut username, password, dan email.
Menerapkan enkapsulasi dengan metode login dan ganti password.
Status / Perbaikan:
Kode sudah benar setelah memperbaiki operator >=, menampilkan hasil validasi akun.
9. Kelas RekeningBank
Penjelasan Kelas:
