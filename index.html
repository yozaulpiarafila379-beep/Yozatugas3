Judul : Pembelajaran Konsep Constructor dan Enkapsulasi Melalui Implementasi Berbagai Kelas dalam Dart

1. Kelas Buku
Penjelasan Kelas: Kelas ini merepresentasikan objek buku dengan atribut judul. Constructor digunakan untuk memberikan nilai judul saat objek dibuat.
Perbaikan: Penulisan class harus class Buku dan atribut String judul.

Hasil Program:

2. Kelas Mobil
Kelas mobil memiliki atribut merk dan tahun. Menggunakan parameterized constructor dan named constructor.
3. Kelas Warna
Kelas ini menggunakan const constructor dengan atribut final r, g, b sehingga objek tidak dapat diubah setelah dibuat.
4. Kelas Titik
Constructor menggunakan initializer list untuk memproses nilai sebelum objek dibuat.
5. Kelas Mahasiswa
Memiliki constructor default dan named constructor dariMap untuk membuat objek mahasiswa.
6. Kelas Person
Menerapkan konsep enkapsulasi menggunakan atribut private _name dengan getter dan setter.
7. Kelas Produk
Menggunakan atribut private dan setter dengan validasi nama minimal 3 karakter dan harga harus lebih dari 0.
8. Kelas User
Kelas user menggunakan enkapsulasi untuk username, password dan email dengan validasi tertentu.
9. Kelas RekeningBank
Kelas ini mengelola data rekening bank dengan atribut private _nomorRekening, _saldo, dan _pemilik. Metode setor dan tarik digunakan untuk mengelola saldo.

Kesimpulan

Constructor digunakan untuk menginisialisasi objek ketika dibuat. Enkapsulasi digunakan untuk melindungi data dengan membuat atribut private serta menyediakan getter dan setter untuk mengaksesnya.
